An eCommerce toolkit used to show featured or exclusive products first priority basis in WooCommerce shop.

== Description ==

WC Featured Products is a toolkit, based on WooCommerce where you can display your products always on top by making it a featured product.

Just activate it and listed your product first in shop page-

== Installation ==

= Minimum Requirements =

* PHP 5.6 or greater is recommended

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handles the file transfer, and you won’t need to leave your web browser. To do an automatic install of WC Featured Products, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type “WC Featured Products,” then click “Search Plugins.” Once you’ve found us, you can install it by! Clicki “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the WC Featured Products plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Frequently Asked Questions ==

= Default settings =

When activated this plugin will enable global settings. It shows anywhere featured products at top priority. These settings can be changed from plugin's settings.

= Is this plugin compatible with WooCommerce? =

Yes, absolutely.

= How can i set featured product? =

To set featured product. Please see woocommerce documentation [Mark a product as Featured](https://docs.woocommerce.com/document/managing-products/#section-20) for help.
